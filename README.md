# Mini Turtle Task

Un paquete educativo en Python que simula los movimientos de una tortuga usando dibujo en texto (ASCII).
Permite realizar acciones simples como avanzar, bajar y reiniciar la posición horizontal.

Este proyecto se creó con el objetivo de practicar:
- La estructuración de paquetes en Python
- El uso y comprensión de variables globales
- La separación de la lógica y la interfaz
- El manejo de imports entre archivos
